# cURL Example

This is an example cURL command for accessing the U-M GPT Toolkit API.

## Setup

Fill in your credentials in curl.sh for API_KEY, API_BASE, and API_MODEL.

| Variable | Description |
|----------|-------------|
| `API_KEY` | Your API key |
| `API_BASE` | The gateway base URL |
| `MODEL` | Model to be used |

Run `Python/get_models.py` to see all available model names.
